# How to test it

## Demo & Test

Clone this git repository:
```sh
git clone git@github.com:pixari/vue-i18n-extract.git
```

```sh
// install dependencies:
npm install
// compile typescript:
npm run build
// this will use the data in the demo folder to generate a report.
npm run demo
```

or 

```sh
// install dependencies:
yarn install
// compile typescript:
yarn build
// this will use the data in the demo folder to generate a report.
yarn demo
```

To run tests:

```sh
npm run test
```

or

```sh
yarn test
```
